
拉取项目
----------------------------

VS打开控制台  ctrl ~

git clone 项目

npm install
git branch   新建分支
git branch -v  查看所有分支
mpm run serve   启动项目

如果npm安装有问题  
1）那么删除node_modules文件夹  
2）git clean -xdf
3）npm install
